# PowerGym - Aplicativo Web para Gerenciamento de Academia

Bem-vindo ao PowerGym, um aplicativo web moderno projetado para simplificar os processos de gerenciamento de academias. Este repositório contém o código-fonte para o aplicativo web PowerGym, construído utilizando Next.js, Tailwind CSS, Zod e React Hook Form.

## Funcionalidades

- **Interface Amigável**: Interface intuitiva projetada para administradores e membros da academia.
- **Gerenciamento Eficiente**: Simplifique tarefas de gerenciamento da academia, como cadastro de membros, agendamento e acompanhamento de progresso.
- **Estilo Personalizável**: Utiliza Tailwind CSS para estilização personalizável, permitindo uma aparência moderna e profissional.
- **Validação de Dados**: Integração com Zod para validação robusta de dados, garantindo integridade e segurança dos dados.
- **Manipulação de Formulários**: Utiliza React Hook Form para manipulação eficiente e flexível de formulários.

## Tecnologias Utilizadas

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Zod](https://github.com/colinhacks/zod)
- [React Hook Form](https://react-hook-form.com/)

## Como Começar

1. **Clone o Repositório**:
   ```bash
   git clone https://github.com/seu-usuario/powergym.git
   ```

2. **Instale as Dependências**:
   ```bash
   cd powergym
   npm install
   ```

3. **Execute o Aplicativo**:
   ```bash
   npm run dev
   ```
   O aplicativo estará rodando em `http://localhost:3000`.
